Can 与 Be Able To 之间的异同分析

Posted on 2020-05-09

概述

$Can$ 和 $be$ $able$ $to$ 通常是可以互换的，并且可以 通常 使用其中一个而含义没有差异，但并不总是这样。

$Can$ 是表示现在时的一般能力的模态助动词，或者可以表示过去时的一般能力的模态助动词。

$Be$ $able$ $to$ 不是模态助动词，只是动词 $’$ be $’$ $ + $ 副词$’$ able $’$ $ + $ 不定式$’$ to $’$ 。

现有能力$Present$ $Ability$

既可以使用 $can$ 也可以使用 $be$ $able$ $to$ ，但是一般而言 $can$ 的使用的更为常见：

  • $a.$ James can speak three languages.

  • $b.$ James is able to speak three languages.

  • $c.$ Michelle can easily multitask.

  • $d.$ Michelle is able to easily multitask.

未来能力$Future$ $Ability$

只会使用 $will$ $be$ $able$ $to$ 来谈论尚未拥有的未来技能或能力，但是将来会拥有。 绝不会将 $can$ 用作将来只会拥有的功能：

  • $a.$ When I finish training I will be able to run a 5 minute mile. $(correct)$
  • $b.$ When I finish training I can run a 5 minute mile. $(wrong)$
  • $c.$ I will be able to see better when I get new glasses. $(correct)$
  • $d.$ I can see better when I get new glasses. $(wrong)$

决策或未来安排$Decisions$ $and$ $Future$ $Arrangements$

在讨论决策和未来安排时，可以使用 $can$ 或 $be$ $able$ $to$：

  • $a.$ The doctor can see you next Monday.
  • $b.$ The doctor is able to see you next Monday.

过去的能力$Past$ $Ability$

可以使用 $can$ 或 $be$ $able$ $to$ 谈论某种过去的能力，但这种能力现在已不存在：

  • $a.$ When I was young, I was able to eat like a pig and not get fat.
  • $b.$ When I was young, I could eat like a pig and not get fat.

与单个时间相关的能力$Ability$ $Related$ $to$ $a$ $single$ $event$

  • $a.$ We were able to go sailing yesterday because the weather was so nice. $(correct)$
  • $b.$ We could go sailing yesterday because the weather was so nice. $(wrong)$

但是，可以使用 $can$ 或 $was/were$ $able$ $to$ 能够处理一些陈述性动词（见，听，觉，品味）来谈论与单个事件相关的能力：

  • $a.$ Were you able to see the fireworks from your balcony last night?
  • $b.$ I was able to taste the salt in her cooking.
  • $c.$ I could taste the salt in her cooking.

其他情况$Other$ $Situations$

对于单个事件和很长一段时间，使用 $couldn’t/wasn’t$ 或 $weren’t$ $able$ $to$ 做出否定性陈述：

  • $a.$ Yesterday he wasn’t able to finish his dinner.
  • $b.$ Yesterday he couldn’t finish his dinner.
  • $c.$ I wasn’t able to swim when I was younger.
  • $d.$ I couldn’t swim when I was younger.
